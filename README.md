# Reaction Timer Game

A simple one-page reaction timer game built with HTML, CSS, and JavaScript.

## How it works

- The game starts with 10 lights arranged in a grid.
- One light turns on at a time.
- Click the light as quickly as possible to turn it off.
- The game records the total time it takes to turn off all 10 lights.
- The final score is the sum of all reaction times.

## Usage

1. Open `index.html` in a web browser.
2. Press the **Start Game** button.
3. Click the lit square as soon as it turns on.
4. Repeat until all 10 lights are turned off.
5. Your total score appears when the game is complete.
